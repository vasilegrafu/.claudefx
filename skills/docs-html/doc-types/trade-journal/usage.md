# trade-journal

- Audience: the trader — the journal exists to make behavior visible, not to
  look good.
- Filename: `docs/trade-journal-<period>.html` (e.g. `-2026-h2`), one file per
  period; a new subsection per trade.
- Depth: none — entries are written by the owner at trade time.
- Rules: every trade records the PLAN (entry/stop/target) and the DEVIATION
  from it — a profitable trade that broke the rules is logged as a failure of
  process; lessons must end as rule-change candidates for the Standing rules
  section or be dropped; never edit past entries (append corrections).
