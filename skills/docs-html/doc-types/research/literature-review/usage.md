# literature-review

- Audience: a researcher entering the topic — the map of who said what.
- Filename: `docs/literature-review-<topic>.html`.
- Depth: ask — full means actually reading the sources.
- Rules: sources are COMPARED, not summarized in sequence — the table forces
  it; disagreements between sources are the interesting part, explain them;
  every source in the table appears in References with a full citation.
