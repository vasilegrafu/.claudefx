# due-diligence-report

- Audience: the decision maker (often future you) verifying a company, fund,
  or counterparty before committing money.
- Filename: `docs/due-diligence-report-<subject>.html`
- Depth: ask — full diligence reads primary sources (filings, accounts).
- Rules: every factual claim traces to a reference; red flags are stated even
  when the verdict is positive; SWOT entries are specific to the subject; the
  assessment ends in an explicit verdict, never a shrug.
