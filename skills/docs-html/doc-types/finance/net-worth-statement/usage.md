# net-worth-statement

- Audience: the owner — the personal balance sheet, taken on a fixed cadence.
- Filename: `docs/net-worth-statement-<date>.html` (or one living document
  with change-history).
- Depth: none.
- Rules: every asset states its valuation basis (market, estimate, cost) —
  mixing them silently inflates; illiquid assets flagged; same line items
  every period so the trend means something; one currency, conversions noted.
