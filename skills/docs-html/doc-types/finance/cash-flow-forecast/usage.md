# cash-flow-forecast

- Audience: whoever must not run out of cash — the planning twin of budget
  (budget allocates, forecast times).
- Filename: `docs/cash-flow-forecast-<scope>-<horizon>.html`.
- Depth: none — the owner's numbers.
- Rules: periods granular enough to expose the tightest point (monthly at
  least); closing cash of one period is opening of the next — the table must
  chain; the stress view is mandatory: a forecast that only works when
  everything lands on time is a wish.
