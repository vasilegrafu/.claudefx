# valuation-report

- Audience: whoever acts on the number — buy, sell, negotiate, or account.
- Filename: `docs/valuation-report-<asset>-<date>.html`.
- Depth: ask.
- Rules: a valuation is a RANGE with a method, never a point with an aura;
  every input carries its basis (market data, comparable, judgment); the
  sensitivity table is mandatory — it shows how much the answer is
  assumption; formulas via `c.formula()` (KaTeX), computed values stated.
