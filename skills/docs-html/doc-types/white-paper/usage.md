# white-paper

- Audience: outside readers you want to persuade — the formal position piece.
- Filename: `docs/white-paper-<topic>.html`.
- Depth: full — a white paper is finished writing, not a skeleton.
- Rules: one thesis per paper; the strongest counter-argument is addressed in
  Evidence, not ignored; implications are actionable statements; formal tone,
  ISO front/back matter conventions apply (lead, references).
