# investment-policy-statement

- Audience: the investor's calm self — the document that overrules the
  emotional self in drawdowns and euphoria; or a client relationship.
- Filename: `docs/investment-policy-statement.html` (one; versioned via
  change-history).
- Depth: full — this is a formal governing document.
- Rules: rules are `requirement` cards with IPS- trace ids, verifiable by a
  third party; allocation carries min/max bands, not just targets; changes go
  through `modify` with version bump — mid-drawdown edits deserve a
  revision-note explaining why the change is not capitulation.
