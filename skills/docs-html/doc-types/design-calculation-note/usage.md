# design-calculation-note

- Audience: a checker — the classic calc sheet, written so an independent
  engineer can verify every step.
- Filename: `docs/design-calculation-note-<subject>.html`.
- Depth: full.
- Rules: every number carries units; assumptions are separated from givens
  (an assumption can be wrong, a given is contractual); the result includes a
  check by an independent method; Pass/Fail verdict via badge — never
  implied.
