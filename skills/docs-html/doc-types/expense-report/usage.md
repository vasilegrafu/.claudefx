# expense-report

- Audience: the approver and bookkeeping.
- Filename: `docs/expense-report-<claimant>-<period>.html`.
- Depth: none.
- Rules: every line has a date and category; receipts referenced (numbered)
  even when stored elsewhere; one currency per claim (convert and note the
  rate otherwise); the approval block is the point — unapproved claims stay
  v0.x.
