# bill-of-materials

- Audience: purchasing and assembly — the list that must be exactly right.
- Filename: `docs/bill-of-materials-<assembly>.html`.
- Depth: none.
- Rules: the BOM matches a named design revision (state it); every line has a
  verifiable spec or part number — "suitable bolt" is not a part; quantities
  include units; substitution rules explicit (allowed equivalents or "no
  substitution").
