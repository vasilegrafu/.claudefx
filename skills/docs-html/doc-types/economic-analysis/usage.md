# economic-analysis

- Audience: whoever's decision depends on the macro picture (often the
  investor's own market-outlook and theses).
- Filename: `docs/economic-analysis-<economy-or-sector>-<period>.html`.
- Depth: ask.
- Rules: indicators cite official/primary sources; the Analysis explains the
  numbers rather than restating them; scenarios carry probabilities; the
  Implications section names the decision this analysis feeds — analysis
  without a consumer is a hobby.
