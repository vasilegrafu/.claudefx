# financial-statements

- Audience: the entity's owner, partner, or accountant — the period's formal
  picture.
- Filename: `docs/financial-statements-<entity>-<period>.html`.
- Depth: none — figures come from the books.
- Rules: statements must tie (balance sheet balances; cash flow reconciles to
  cash movement); comparative prior column always present; negatives in
  parentheses via `.neg`; anything surprising gets a Note — statements answer
  questions before they are asked.
