# proposal

- Audience: the client deciding whether to buy — the commercial sibling of
  statement-of-work (the SOW formalizes what a won proposal promised).
- Filename: `docs/proposal-<client>-<subject>.html`.
- Depth: full.
- Rules: deliverables are verifiable statements, not aspirations; out-of-scope
  is explicit (scope disputes are born here); price table totals; a validity
  date always — open-ended offers rot.
