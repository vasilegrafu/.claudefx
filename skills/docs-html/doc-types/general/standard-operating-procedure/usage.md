# standard-operating-procedure

- Audience: the person performing the task — possibly for the first time,
  possibly at 3 a.m. The general SOP (the software runbooks are its siblings).
- Filename: `docs/standard-operating-procedure-<task>.html`.
- Depth: full.
- Rules: every step is ONE action with its expected result (the performer must
  know they're still on track); prerequisites checked before step 1, not
  discovered at step 7; the escalation path has a name/contact, not "someone".
