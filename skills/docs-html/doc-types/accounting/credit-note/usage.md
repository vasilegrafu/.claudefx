# credit-note

- Audience: the client and both parties' bookkeeping — the legal way to
  correct an issued invoice (invoices are immutable).
- Filename: `docs/credit-note-<number>.html` (own sequential numbering).
- Depth: none.
- Rules: always references the invoice it corrects; amounts negative
  (parenthesized `.neg`); the reason stated; mirrors the invoice's mandatory
  legal fields; like invoices, an issued credit note is immutable.
