# policy-brief

- Audience: a decision maker with five minutes — the short, actionable form
  (economic-analysis is its long-form evidence base).
- Filename: `docs/policy-brief-<issue>.html`.
- Depth: ask.
- Rules: two pages' worth, maximum; "do nothing" is always an option in the
  table; the recommendation names its success measure; every number in the
  issue section has a source.
