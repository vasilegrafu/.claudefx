# industry-analysis

- Audience: whoever must act on the industry's structure — investors (feeds
  due-diligence and theses), entrants, strategists.
- Filename: `docs/industry-analysis-<industry>.html`.
- Depth: ask.
- Rules: structure with numbers (shares, sizes, sources); the five forces
  each get a verdict with a reason, not a rating alone; dynamics name WHO the
  change favors; implications name the consuming decision.
