# earnings-note

- Audience: the holder of the position, quarterly — the thesis check-in.
- Filename: `docs/earnings-note-<ticker>-<quarter>.html`.
- Depth: none — written from the report/call, same week.
- Rules: graded against the INVESTMENT-THESIS pillars, verbatim — not against
  the market's mood; every note ends in an explicit action (hold is an
  action); a Broken pillar forces a thesis `modify` or an exit, never a
  silent shrug.
