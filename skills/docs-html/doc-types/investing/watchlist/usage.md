# watchlist

- Audience: the investor's future self at the moment a trigger fires.
- Filename: `docs/watchlist.html` (one living document, change-history).
- Depth: none.
- Rules: every candidate has a TRIGGER with a level — a watchlist without

  investment-thesis before any order; dropped entries stay listed with the
  reason (the graveyard teaches).
