# commissioning-report

- Audience: the party accepting the installation — the record that it works
  as specified, witnessed.
- Filename: `docs/commissioning-report-<installation>.html`.
- Depth: none — written during commissioning.
- Rules: every test traces to a spec requirement; measured values recorded
  even when passing; deviations dispositioned explicitly (accepted needs the
  accepter's name); the approval block is the handover moment.
