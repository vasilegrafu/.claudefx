# inspection-report

- Audience: whoever must accept or reject the inspected item — and auditors
  later.
- Filename: `docs/inspection-report-<subject>-<date>.html`.
- Depth: none — written at inspection time.
- Rules: every check names the requirement it verifies (traceable to the
  spec); observed values recorded even when passing; a report with any Fail
  must end in an explicit disposition decision; findings are facts —
  interpretation lives in the disposition.
