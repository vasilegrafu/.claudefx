# portfolio-review

- Audience: the portfolio owner, reviewing on a fixed cadence (monthly or
  quarterly) — discipline against drift and narrative-chasing.
- Filename: `docs/portfolio-review-<period>.html` (e.g. `-2026-q2`).
- Depth: draft-friendly — the numbers come from the owner's own systems.
- Rules: the SAME KPI set every period (comparability beats richness); every
  action item is sized and dated; the thesis-status column forces re-reading
  each position's investment-thesis; the performance chart is a mermaid
  xychart — data lives in the document, editable like any diagram.
