# swot-grid

_Authoring guidance for the `swot-grid` component — when to use it, how, and the rules._

Styled by: `css/domain-specific/business.css`

The classic 2×2: Strengths | Weaknesses over Opportunities | Threats
(internal factors top, external bottom), color-coded quadrants.

`{{ c.swot_grid(strengths=[...], weaknesses=[...], opportunities=[...], threats=[...]) }}`

Rules: 2–5 bullets per quadrant; every entry is specific to the subject (if it
could be said of any company, delete it).
