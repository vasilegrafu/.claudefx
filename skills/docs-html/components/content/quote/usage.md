# quote

Styled by: `css/modules/content.css`

A pull-quote: cited words set apart from the prose, with attribution.

`{% call c.quote(source="Nygard, 2011") %}…the quoted words…{% endcall %}`

Rules: quotes are VERBATIM (edits in [brackets]); always attribute (source
appears in References when it is a real citation); use sparingly — a document
of quotes is a collage, not an argument.
